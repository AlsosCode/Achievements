# Pair Extraordinaire Achievement

This commit demonstrates co-authoring to unlock the Pair Extraordinaire achievement!

## How it works:
Add `Co-authored-by: Name <email@example.com>` to your commit messages.

## Example:
```
git commit -m "Your commit message

Co-authored-by: Partner Name <partner@example.com>"
```

You need to co-author commits with another GitHub user to unlock this achievement.
